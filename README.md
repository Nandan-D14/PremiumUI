# Premium UI Component Library

A premium collection of animated React components built with Tailwind CSS and Framer Motion. Copy, paste, and customize.

## Features

*   **44+ Premium Components**: Buttons, Cards, Inputs, Avatars, Alerts, Selects, and more.
*   **Framer Motion Animations**: Smooth, 60fps animations out of the box.
*   **Tailwind CSS**: Fully customizable with utility classes.
*   **TypeScript**: Fully typed for a great developer experience.
*   **Dark Mode First**: Designed for modern, dark-themed interfaces.
*   **Copy & Paste**: No npm install required. Just copy the code you need.

## Component Categories

### UI Components
Avatar, Switch, Skeleton, Divider, Alert, Select, Button, Input, Badge, Card, Tabs, Accordion, Modal, Tooltip, Progress, Glass Card, Star Border, Spotlight Card, Property Showcase Card

### Text Animations
Split Text, Typewriter Text, Decrypted Text, Gradient Text

### 3D & Interactive
Tilted Card, Glowing Card, Animated Gradient, Shimmer Button, Parallax Card, Dock

### Backgrounds
Nebula Swirl

### Templates
Complete landing pages and dashboards including Modern SaaS, Spotify Dashboard, Nebula Portfolio, and more.

## Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/Nandan-D14/Premium-UI.git
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run dev
    ```
